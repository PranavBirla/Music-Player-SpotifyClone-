Spotify Frontend Clone

A fully responsive Spotify frontend clone built with HTML, CSS, and JavaScript, showcasing my skills in creating interactive, visually appealing web applications.

Features

Responsive Design – Works smoothly on desktop, tablet, and mobile.

Interactive Music Player – Play/pause, progress bar, and volume control.

Smooth Animations – Modern CSS transitions for a clean user experience.

Organized Codebase – Built with core web technologies for clarity and scalability.

Tech Stack

HTML5 – For structured layout.

CSS3 – For styling, responsiveness, and animations.

JavaScript (Vanilla) – For interactivity and DOM manipulation.

Purpose

This project demonstrates my ability to:

Build responsive UIs without frameworks.

Implement interactive features using pure JavaScript.

Maintain clean, modular, and well-structured code.

How to Run Locally

Clone this repository:

git clone https://github.com/your-username/spotify-clone.git


Open index.html in your browser.
